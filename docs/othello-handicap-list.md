# オセロ ハンデ一覧（実戦用）

実力差がある相手同士でも楽しく対戦できるように、使いやすいハンデを強さ別にまとめた一覧です。

## 1. まず使うおすすめハンデ（簡単）

1. 弱い側が先手固定
- 初心者救済として最も簡単。
- 1段階の軽いハンデ。

2. 強い側は角打ち禁止（初回のみ）
- 強い側は最初に角を取れる手があっても1回だけ禁止。
- 2段階の軽〜中ハンデ。

3. 弱い側に「3手だけ合法手ヒント」
- 対局中に3回だけ、打てる場所を表示してよい。
- 学習効果が高い中ハンデ。

## 2. 定番ハンデ一覧（強さ順）

### レベル1（軽いハンデ）

- 先手固定（弱い側）
- 持ち時間ハンデ（強い側を短くする）
  - 例: 弱い側 10分、強い側 6分
- 強い側は序盤5手まで「X打ち（角隣接危険マス）」禁止

### レベル2（中ハンデ）

- 強い側は「角取り」1回パス
  - 角を取れる局面でも別手を選ぶ
- 初期配置をずらす（弱い側に有利な形）
  - 弱い側が角に近い安定石を作りやすい配置にする
- 弱い側に2回だけ「打ち直し権」
  - 置いた直後なら別の合法手に変更可（相手が打つ前まで）

### レベル3（重いハンデ）

- 初期石ハンデ
  - 開始前に弱い側の石を2〜4個追加
- 強い側は偶数手のみ思考時間10秒制限
- 強い側は終盤10手で「角打ち禁止」

## 3. 実力差ごとの推奨セット

目安として、普段の勝率差や経験差で選ぶ。

### 実力差 小（勝率55:45〜65:35）

- 先手固定（弱い側）
- 強い側の持ち時間を 20〜30% 短縮

### 実力差 中（勝率65:35〜80:20）

- 先手固定（弱い側）
- 強い側の角取り1回パス
- 弱い側に合法手ヒント 2〜3回

### 実力差 大（勝率80:20以上）

- 先手固定（弱い側）
- 初期石ハンデ 2〜4個
- 強い側の持ち時間を半分
- 弱い側に打ち直し権 1〜2回

## 4. 初期石ハンデの置き方（例）

公平性を崩しすぎないよう、追加位置は固定テンプレートを推奨する。

### 追加2個（軽め）

- 弱い側の色を `B` とする場合、`c4` と `f5` に追加
- 先後で色が変わるなら、常に「弱い側の色」で同じ形を使う

### 追加4個（重め）

- `c4`, `f5`, `c5`, `f4` に弱い側の石を追加
- 中央付近の安定化を補助し、序盤の事故を減らす

## 5. オンライン対戦での運用ルール

- 対局開始前にハンデを全員へ明示
- ルーム名にハンデを記載
  - 例: `ハンデ: 先手固定 + 角1回パス`
- 対局ログにハンデ設定を保存
- 再戦時はハンデを1段階だけ見直す

## 6. NGハンデ（非推奨）

- 「強い側はランダムに打つ」
  - ゲーム性が壊れやすい
- 「角を永久禁止」
  - 終盤の読み合いを損ないやすい
- 「弱い側だけパス可能」
  - ルールの一貫性が崩れる

## 7. すぐ使えるプリセット

### プリセットA（初心者と経験者）

- 先手: 弱い側
- 強い側: 角取り1回パス
- 弱い側: 合法手ヒント3回

### プリセットB（実力差がかなり大きい）

- 先手: 弱い側
- 初期石ハンデ: 2個
- 持ち時間: 弱い側10分 / 強い側5分

### プリセットC（短時間で盛り上がる）

- 先手: 弱い側
- 強い側: 序盤5手までX打ち禁止
- 持ち時間: 弱い側5分 / 強い側3分
