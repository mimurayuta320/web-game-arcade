# 大富豪 追加ルール表

この表は、現在このプロジェクトで設定できる大富豪ルールの一覧です。

## 基本ルール

| ルール名 | 発動条件 | 効果 | 備考 |
| --- | --- | --- | --- |
| 革命（強さ反転） | 常時 | 強さが反転（通常: 3 < ... < A < 2、革命時: 2 > ... > 3） | 11バックと重なると都度反転状態を計算 |
| 8切り | 8を出した時 | 場を流し、出した人が続けて手番 | 上がりにも対応 |
| 2上がりで場流し | 2を出した時 | 場を流し、出した人が続けて手番 | ON時のみ |
| ジョーカーあり | 常時 | ジョーカーをデッキに追加 | 通常札より強い扱い |
| スペ3返し | 場がジョーカーで、♠3を出した時 | 場を流し、出した人が続けて手番。ジョーカー側は次回手番を1回スキップ | ON時のみ |
| 6リバース | 6を出した時 | 手番方向が反転 | 2人戦では体感差は小さめ |
| 4止め | 4を出した時 | 場を流し、出した人が続けて手番 | ON時のみ |
| 9リバース | 9を出した時 | 手番方向が反転 | 2人戦では体感差は小さめ |
| 12ボンバー | Q(12)を出した時 | 場を流し、出した人が続けて手番 | ON時のみ |
| A流し | Aを出した時 | 場を流し、出した人が続けて手番 | ON時のみ |
| Kリバース | Kを出した時 | 手番方向が反転 | 2人戦では体感差は小さめ |
| 3スキップ | 3を出した時 | 次プレイヤーの手番を飛ばす | 2人対戦では実質的に連続手番 |
| 前回勝者が次ゲーム先攻 | 新ゲーム開始時 | 前ゲームの勝者が先攻になる | ON時のみ |
| 勝者敗者でカード交換（強⇔弱） | 新ゲーム開始時 | 前ゲーム勝者の最弱札と、敗者の最強札を交換 | ON時のみ |
| ド田舎ルール | 常時 | 防衛失敗で次ゲームに交換ペナルティ | 詳細: docs/daifugo-doinaka-spec.md |
| J流し | J(11)を出した時 | 場を流し、出した人が続けて手番 | ON時のみ |
| Qリバース | Q(12)を出した時 | 手番方向が反転 | 12ボンバーと併用可能 |
| パス禁止（出せる時は必ず出す） | 常時 | 出せるカードがある状況ではPASS不可 | 出せない時のみPASS可能 |
| 連続出し（同ランク複数枚） | 同ランクを持っている時 | 1枚出した直後に同ランクを追加で複数枚出せる | 最大枚数を2〜4で設定可能 |
| 色しばり（赤/黒） | 同色連続で出した時 | 赤または黒で場流れまで縛る | ジョーカーは色なし扱い |
| 5スキップ | 5を出した時 | 次プレイヤーの手番を飛ばす | 2人対戦では実質的に連続手番 |
| 7渡し | 7を出した時 | 出した人の手札から好きな1枚を相手へ渡す | CPU時は自動選択 |
| 10捨て | 10を出した時 | 出した人の手札から好きな1枚を追加で捨てる | CPU時は自動選択 |
| 11バック | 11を出した時 | 場が流れるまで強さを反転 | 11を再度出すと元に戻る |
| 上がり禁止（2 / 8） | 最後の1枚が2または8 | 8上がりは常に反則、2上がりは非革命時のみ反則 | ON時のみ |
| ジョーカー上がり禁止 | 最後の1枚がジョーカー | 反則上がりとして相手の勝利 | ON時のみ |
| 反則上がり拡張（追加ランク指定） | 指定したランクで上がり | 反則上がりとして相手の勝利 | 例: A,J,Q,K,JK |
| しばり（同じマークのみ） | 場にカードがある間 | 同じスートのみ出せる | 連続しばりOFF時 |
| 連続しばり（同スート連続で発動） | 同スートが連続した時 | しばり状態を固定して場流れまで継続 | 「しばり」と併用前提 |
| 開始は3♦持ちから | ゲーム開始時 | 3♦を持つプレイヤー先攻 | OFF時はランダム |

## 追加カスタムルール

| ルール名 | 発動条件 | 効果 |
| --- | --- | --- |
| 指定ランク場流し | 指定ランクを出した時 | 場を流して同一プレイヤーが続行 |
| 次手ランク制限 | 指定ランクを出した次の手番 | 次プレイヤーが指定ランクのみ出せる |

## 適用範囲

- PvP（2P LOCAL）: 基本ルール + 追加カスタムルールの両方を適用可能。
- CPU対戦: クラシックルール（基本ルールのみ）を選択・適用可能。
- ルーム対戦: クラシック固定（ルール編集・適用は不可）。
