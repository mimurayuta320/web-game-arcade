# Fit Puzzle 拡張仕様書

更新日: 2026-07-18
対象: Neon Fit Puzzle

## 1. 目的
Fit Puzzle を単発クリア型から、継続的に攻略したくなるステージ進行型へ拡張する。

## 2. 実装サマリ
- ステージ個性付きの 100 ステージ構成を追加
- スコア / タイムを HUD に追加
- クリア時ランク評価 (S/A/B/C) と加点表示を追加
- 1 ステージにつき 1 回使える ASSIST スキルを追加
- 盤面列数を可変化し、6x6 以外の盤面に対応
- ステージごとの正解配置を固定化（同条件なら毎回同じ形）

## 3. UI 仕様
### 3.1 HUD 要素
- BOARD: 現在ステージ番号、総ステージ数、ステージ名、盤面サイズを表示
- PLACED: 配置済みピース数 / 総ピース数
- SCORE: 合計スコア
- TIME: 現在ステージの経過時間 (mm:ss)
- DIFFICULTY: EASY / NORMAL / HARD を選択

### 3.2 ボタン
- GAME START:
  - 待機中: ステージ1開始
  - プレイ中: 現在ステージを再生成して開始
- NEXT STAGE:
  - クリア後のみ有効
  - 次ステージへ進行（最終ステージクリア後はステージ1へ）
  - 未開始または未クリア時は進行不可メッセージを表示
- ROTATE: 選択中ピースを90度回転
- NO ROTATION ON/OFF:
  - 回転なしモードの切替
  - ON時はROTATEボタン無効
  - ON/OFF切替時、プレイ中なら現在ステージを再開始して即反映
- ASSIST x1/x0:
  - 各ステージで1回のみ利用可能
  - 選択ピースを置ける推奨位置を盤面ハイライト
- RESET: 現在ステージを初期化
- MENU: メニューへ戻る確認ダイアログ

### 3.5 難易度モード
- EASY:
  - ピース数を減らす
  - いびつ形状条件（mutation / complex / branch）を緩和
  - ASSIST 回数を増やす
- NORMAL:
  - 基本ステージ設定
- HARD:
  - ピース数を増やす
  - いびつ形状条件を強化
  - ASSIST 回数を減らす
- プレイ中に難易度を変更した場合は現在ステージを再開始して反映

### 3.3 ドラッグアンドドロップ操作
- ピース一覧から盤面セルへドラッグして配置できる
- 盤面上の配置済みピースをドラッグして別セルへ移動できる
- 配置済みピースをピース一覧へドロップすると手元へ戻せる
- 無効な位置へドロップした場合、元の位置へ自動復帰する
- ポインタイベントベースでドラッグ操作を処理（PC / タッチ共通）
- 盤面の配置済みブロックをダブルクリックすると、そのピースを手元へ戻せる

### 3.4 回転なしバージョン
- 回転なしモードON時は、全ピースを回転0の状態で生成
- 回転なしモードON時はROTATE操作を受け付けない
- ステージ表示メッセージに [回転なし] を付与

## 5. ピース生成仕様
- 初期状態は盤面全体を長方形分割して生成
- その後、隣接ピース間でセルを再配分し、連結性を保ったまま変形ピースへ変換
- 各ピースは常に1つの連結成分を維持（分断不可）
- ステージごとの profile 条件（mutationSteps, minComplex, minBranch）を満たすまで最大140回再生成
- 条件を満たせない場合は最後の生成結果を採用
- ステージ後半ほど mutationSteps / minComplex / minBranch を増やし、難易度を段階的に上げる
- 生成乱数はステージごとにシード固定し、同一ステージ・同一難易度・同一モードでは同じ正解配置になる

### 5.1 用語
- 複雑形状 (complex): ピース面積 < 外接矩形面積 の形状
- 分岐形状 (branch): 同一ピース内で隣接セル数が3以上のセルを持つ形状

## 6. スコア仕様
### 6.1 ステージ最大点
maxScore = rows * cols * 20 + pieceCount * 120

### 6.2 ペナルティ
penalty = moveCount * 7 + failCount * 25 + elapsedSec * 2 + assistUsedCount * 140

### 6.3 ステージ獲得点
stageScore = max(120, maxScore - penalty)

### 6.4 合計点
totalScore = 各ステージ stageScore の合計

## 7. ランク仕様
score / maxScore 比率で判定
- S: 0.85 以上
- A: 0.70 以上
- B: 0.50 以上
- C: それ未満

## 8. ASSIST 仕様
- 使用回数: ステージごとに設定（前半 1 回、後半 0 回）
- 選択ピースが未選択または配置済みの場合、未配置ピースを自動選択
- 置ける位置がある場合:
  - 推奨範囲セルをハイライト表示
  - ボタン残回数を 1 ずつ減算
- 置ける位置が無い場合:
  - メッセージ表示のみ、使用回数は消費しない

## 9. 内部状態（主要）
- stageIndex: 現在ステージ
- stageTitle: ステージ表示名
- moveCount: 成功配置回数
- failCount: 置けなかった試行回数
- elapsedSec: 現在ステージ経過秒
- totalScore: 累積スコア
- assistRemaining: 現在ステージのASSIST残回数
- assistUsedCount: 現在ステージでのASSIST使用回数
- hintKeys: ASSISTでハイライト中のセル集合
